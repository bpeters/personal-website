# Website

Personal website.

## Requirements

- [node v8.0.0+](https://nodejs.org/)
- [npm v5.0.0+](https://www.npmjs.com/)
- [yarn](https://yarnpkg.com/)
- [eslint](http://eslint.org/)
- [mocha](https://mochajs.org/)


**Install**

```shell
$ yarn install
```


**Start**

```shell
$ yarn start
$ yarn start -- --release
$ yarn start -- --no-hmr
```


**Test**

```shell
$ yarn lint
$ yarn test
```


**Build**

```shell
$ yarn build
```


**Deploy**

```shell
$ yarn deploy
```
